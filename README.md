# Конфигурационный парсер в YAML

Инструмент командной строки для преобразования текста из учебного конфигурационного языка в YAML формат.

## Особенности

- Парсинг учебного конфигурационного языка
- Поддержка чисел, строк, структур и констант
- Вывод в формате YAML
- Обработка синтаксических ошибок
- Простой интерфейс командной строки

## Требования

- Python 3.6+
- Установленные зависимости: `pip install lark pyyaml`
## Использование 
```
python config_parser.py -i game.conf -o output.yaml
```
## Пример конфигурационного файла: Конфигурация приложения
```
app_name is 'MyApp';
version is '1.0';

struct {
    app = struct {
        name = #(app_name),
        version = #(version),
        settings = struct {
            debug = 'false'
        }
    }
}
```
